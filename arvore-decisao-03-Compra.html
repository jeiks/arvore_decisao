Passos para escolher o atributo a ser utilizado:
✕
  1. Calcula-se a Entropia do Objetivo.
    Exemplo para dois valores, sendo $x$ e $y$:
    $p_x = \frac{|x|}{|x|+|y|}$
    $p_y = \frac{|y|}{|x|+|y|}$
    $E_{Objetivo} = -\sum_i p_i \log_2(p_i)$
    $E_{Objetivo} = - p_x \log_2(p_x) - p_y \log_2(p_y)$
  2. Calcula-se a entropia de cada Atributo existente.
    Nesse momento, deve-se considerar o $x$ e $y$ de cada grupo.
    Exemplo para o Atributo $\mathtt{A}$, com grupos de valores Baixo e Alto.
    O cálculo de $p_{Baixo}$ seria:
    $x_{Baixo} =$ valor Baixo de Atributo $\mathtt{A}$, onde o Objetivo é $x$
    $y_{Baixo} =$ valor Baixo de Atributo $\mathtt{A}$, onde o Objetivo é $y$
    $p_{x_{Baixo}} = -\frac{|x_{Baixo}|}{|x_{Baixo}|+|y_{Baixo}|}$
    $p_{y_{Baixo}} = -\frac{|y_{Baixo}|}{|x_{Baixo}|+|y_{Baixo}|}$
    $p_{Baixo} = -( p_{x_{Baixo}} \log_2(p_{x_{Baixo}}) + p_{y_{Baixo}} \log_2(p_{y_{Baixo}}) )$
  3. Agora, calcula-se a Entropia do Atributo.
    Exemplo para Atributo $\mathtt{A}$:
    $E_A = - \frac{|x_{Baixo}|+|y_{Baixo}|}{|x|+|y|} \times p_{Baixo} - \frac{|x_{Alto}|+|y_{Alto}|}{|x|+|y|} \times p_{Alto} $
  4. Então, calcula-se o Ganho do atributo.
    Exemplo para Ganho do Atributo $\mathtt{A}$:
    $Ganho_A = E_{Objetivo} - E_{A}$
